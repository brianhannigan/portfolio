Surgical Visualization

A 3D interactive surgical environment prototype

Project Overview

This project is a portfolio-friendly demo for showcasing 3D surgical or anatomical visualization in the browser using a Unity WebGL build. It highlights your ability to integrate interactive 3D, host it in GitHub, and present it as part of a professional engineering portfolio.

Features

  • Interactive 3D scene (surgical / anatomical)
  • Unity WebGL export target
  • Mouse/touch camera controls (orbit, zoom)
  • Clean layout for portfolio demos
  • Ready for GitHub Pages / static hosting

Technologies & Tools

  • Unity (with WebGL build support)
  • WebGL
  • HTML / CSS / JS or React wrapper
  • GitHub for hosting

Getting Started

Prerequisites

  • Unity with WebGL module
  • Git
  • Optional: Node.js + npm

Clone

git clone https://github.com/brianhannigan/portfolio.git
cd portfolio/SurgicalVisualization

Build

Open in Unity → Switch build target to WebGL → Build to Build/WebGL/

Run locally

python -m http.server 8000

Then open http://localhost:8000
